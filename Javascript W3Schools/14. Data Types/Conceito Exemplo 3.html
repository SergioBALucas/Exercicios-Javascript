JavaScript lê as expressões da esquerda para direita, A ordem dos fatores produz diferentes resultados:

Neste exemplo, o JavaScript entende 16 e 4 como números, até que ele leia "Volvo", convertendo, assim, toda a expressão como uma string.

Acompanhando em um passo a passo, a variável calcula 16+4, retorna o resultado e o armazena. Após isto, o código lê "Volvo" e transforma tudo em string, e como o código já entendeu 16+4 como 20, o resultado se torna "20Volvo".
